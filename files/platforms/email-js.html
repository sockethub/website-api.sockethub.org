fetch

Email.prototype.fetch = function (job)

Fetches messages from IMAP.  Expects an actor (the account to fetch from) and an object.

The object can be empty, otherwise it can carry these options

perPageNumber of messages to fetch per page.
pagePage number to fetch.
includeBodyInclude message body.

Responds with an object with the following properties

totalTotal number of messages.
countActual number of messages on this page.
perPageAs given in options, or default 10.
pageAs given in options, or default 1.

The actual messages will be sent asynchronously, following the schema of the “send” verb.
